RustDesk
========

Purpose
-------
Use RustDesk for remote access to the Ubuntu box during setup and demo.

Basic steps
-----------
1) Install RustDesk (download from the official site or your internal mirror).
2) Launch RustDesk and set a permanent password.
3) Share the RustDesk ID and password with the team for remote access.

Notes
-----
- Keep the ID/password in a secure channel.
- Disable access when the benchmark or demo is complete.
